# 🔁 Customer Churn Prediction System

A machine learning-powered web application to **predict customer churn** using logistic regression and other models. This project uses a dataset from **Kaggle**, is trained on **Google Colab**, and deployed using **Streamlit** for interactive prediction and visualization.

---

## 📦 Tech Stack

- 💻 **Google Colab** – for training and evaluating the model
- 🌐 **Streamlit** – for building the front-end user interface
- 🧠 **scikit-learn** – for machine learning algorithms
- 📊 **pandas & numpy** – for data manipulation
- 🧾 Dataset from **Kaggle** (e.g., [Telco Customer Churn](https://www.kaggle.com/blastchar/telco-customer-churn))

---

## 📁 Dataset Overview

The dataset typically includes:
- Customer demographics (gender, age, etc.)
- Service usage (internet, phone, streaming)
- Account information (tenure, contract, billing)
- Target: `Churn` (Yes/No)

Example columns:
customerID, gender, SeniorCitizen, Partner, Dependents, tenure, PhoneService,
MultipleLines, InternetService, OnlineSecurity, Contract, MonthlyCharges, TotalCharges, Churn

🚀 How to Run the App Locally

Clone the repository or download the files
Install dependencies:
pip install -r requirements.txt
Run the Streamlit app:
streamlit run app.py
